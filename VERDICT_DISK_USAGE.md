## 📋 ПЛАН ДЕЙСТВИЙ (приоритетный порядок)

### Шаг 1: Быстрая очистка (безопасно, ~1 GB)
```bash
# 1. Очистить npm кэш
npm cache clean --force

# 2. Очистить кэш root
rm -rf /root/.cache/*

# 3. Очистить APT кэш
apt-get clean && apt-get autoclean
```

### Шаг 2: Очистка логов (~2-2.5 GB)
```bash
# Удалить старые ротированные логи
find /var/log -name "*.log.*" -type f -delete

# Проверить большие лог-файлы
find /var/log -type f -size +100M -ls

# Очистить большие логи (оставить последние строки)
# ВНИМАНИЕ: Проверьте каждый файл перед очисткой!
```

### Шаг 3: Детальный анализ /var/www
```bash
# Посмотреть что занимает место
du -sh /var/www/* | sort -h

# Детальный анализ каждого проекта
for dir in /var/www/*; do
    echo "=== $dir ==="
    du -sh "$dir"/*
done | sort -h
```

### Шаг 4: Очистка временных файлов
```bash
# Проверить что в /tmp/judges_documents
ls -lah /tmp/judges_documents

# Удалить старые файлы (старше 7 дней)
find /tmp -type f -atime +7 -delete
```
